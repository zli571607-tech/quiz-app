# 技术栈规范

## 桌面框架
| 技术 | 版本 | 用途 |
|------|------|------|
| Electron | ^33.x | 桌面应用框架，打包 Windows .exe |
| electron-builder | latest | 打包工具，生成安装包 |

## 前端
| 技术 | 版本 | 用途 |
|------|------|------|
| React | ^18.x | UI 框架，构建用户界面 |
| Vite | ^6.x | 构建工具，开发服务器 |
| Tailwind CSS | ^3.x | 样式框架，白色简洁主题 |
| lucide-react | latest | 图标库 |

## 数据库
| 技术 | 版本 | 用途 |
|------|------|------|
| sql.js | ^1.12.x | 纯 JS 的 SQLite 实现，无需编译 |

## AI 接口
| 技术 | 版本 | 用途 |
|------|------|------|
| @anthropic-ai/sdk | latest | Claude API 客户端 |
| Claude 模型 | claude-sonnet-4-6 | 题目生成（性价比高） |

## 文档解析
| 技术 | 版本 | 用途 |
|------|------|------|
| pdf-parse | latest | PDF 文本提取 |
| mammoth | latest | Word .docx 文本提取 |
| pptx-parser | latest | PPTX 文本提取 |
| Node.js fs | 内置 | TXT 文件读取 |

## 开发环境要求
- Node.js >= 20.x
- npm >= 10.x
- Windows 10/11
- Git（可选）
