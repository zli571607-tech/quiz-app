# UI 设计规范

## 设计理念
- **简洁第一**：大量留白，减少视觉负担
- **易用至上**：按钮明确，导航清晰，适合电脑小白
- **快速反馈**：操作后立即有视觉反馈

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | #FFFFFF | 纯白背景 |
| 卡片背景 | #F8F9FA | 浅灰，区分层次 |
| 侧边栏 | #F1F5F9 | 极浅灰蓝 |
| 主色调 | #3B82F6 | 蓝色，按钮和强调 |
| 主色悬停 | #2563EB | 深蓝，hover 状态 |
| 正确色 | #22C55E | 绿色，答对反馈 |
| 错误色 | #EF4444 | 红色，答错反馈 |
| 文字主色 | #1F2937 | 深灰，标题和正文 |
| 文字辅色 | #6B7280 | 中灰，辅助说明 |
| 边框色 | #E5E7EB | 浅灰边框 |

## 字体
- 系统默认字体栈：`"Segoe UI", "Microsoft YaHei", sans-serif`
- 标题：font-semibold，较大字号
- 正文：font-normal，基础字号 15px

## 通用组件风格
- **按钮**：圆角 8px，蓝色填充（主按钮）或蓝色边框（次按钮）
- **卡片**：圆角 12px，白色背景，浅灰边框，轻微阴影
- **输入框**：圆角 8px，边框 #E5E7EB，聚焦时蓝色边框
- **进度条**：圆角，蓝色填充，浅灰底色

## 页面列表

### 1. 首页仪表盘
- 顶部：欢迎语 + 日期
- 中间：统计卡片行（总题数 / 已刷题数 / 平均正确率）
- 下方：快捷操作按钮（导入生成题目 / 开始刷题 / 错题本）

### 2. 导入 & 生成页面
- 大拖拽区域（虚线边框，支持点击选择文件）
- 文件格式提示图标
- 生成设置：题目数量滑块、难度选择
- 生成进度条
- 结果预览列表（可编辑/删除每条题目）
- 确认入库按钮

### 3. 题库管理页面
- 左侧：题套列表（按文档来源分组）
- 右侧：题目列表（分页）
- 每条题目有编辑和删除按钮
- 顶部搜索栏
- 右上角「手动添加」按钮

### 4. 刷题页面
- 居中大卡片展示当前题目
- 4 个选项按钮（A/B/C/D）
- 选择后：正确选项变绿 √，错误选项变红 ✗，同时高亮正确答案
- 底部进度条（第 3/20 题）
- 「上一题」「下一题」按钮

### 5. 成绩页面
- 大圆环显示正确率百分比
- 统计数字：总分 / 正确数 / 错误数 / 用时
- 按钮：查看错题 / 再来一次 / 返回首页

### 6. 错题本页面
- 错题列表，显示题目 + 你的答案 + 正确答案
- 每道题可点击「重做」
- 可「移出错题本」（已掌握）
- 空状态提示：「暂无错题，继续保持！」
