# 需求文档

## 项目目标
开发一款 Windows 桌面刷题软件，用户导入学习文档后，AI 自动生成选择题，支持刷题练习、即时判分、成绩统计和错题回顾。

## 用户故事
1. 用户将学习资料（PDF/Word/TXT/PPT）导入软件
2. 软件调用 AI 自动从文档内容生成选择题（含选项和正确答案）
3. 用户也可以手动添加、编辑、删除题目
4. 刷题时随机出题，用户选择答案后即时判断对错
5. 刷题结束后展示成绩（分数、正确率、用时）
6. 错误的题目自动收录到错题本，方便后续复习重做

## 功能清单

| 编号 | 功能 | 描述 | 优先级 | 状态 |
|------|------|------|--------|------|
| F01 | 文档导入 | 支持 PDF/Word/TXT/PPT 拖拽或选择导入 | P0 | 待开发 |
| F02 | AI 生成题目 | 调用 Claude API 自动生成选择题 | P0 | 待开发 |
| F03 | 手动出题 | 支持手动添加/编辑/删除题目 | P1 | 待开发 |
| F04 | 题库管理 | 按文档分套管理 + 统一总题库 | P0 | 待开发 |
| F05 | 随机刷题 | 随机出题，支持选择套题或全部题库 | P0 | 待开发 |
| F06 | 即时判对错 | 选择后立即显示正确/错误，绿色/红色反馈 | P0 | 待开发 |
| F07 | 成绩统计 | 展示分数、正确率、用时 | P1 | 待开发 |
| F08 | 错题本 | 自动收录错题，支持重做和移除 | P1 | 待开发 |
| F09 | API Key 配置 | 用户填入 Claude API Key | P0 | 待开发 |
| F10 | 软件打包 | 打包为 Windows .exe 安装包 | P2 | 待开发 |

## 非功能需求
- UI 以白色为主，简洁干净
- 操作简单，适合电脑小白使用
- 本地数据存储，无需安装数据库
- 需要联网（AI 功能）
